## 🛠️ 安装运行

```bash
# 克隆项目
git clone <repository-url>
cd review_alarm

# 设置虚拟环境
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/main.py
